# Knowledge Base Handoff

Use this reference after a verified reading loop has passed and discussion memory contains durable insights.

## When To Handoff

Generate a Paper Knowledge Package when:

- the deep read has passed verification;
- source limitations and uncertainties are visible;
- the user has discussed or accepted the main interpretation;
- the user asks to preserve the research for later use; or
- the current work should connect to the long-term research knowledge base.

Do not handoff after a triage brief alone unless the user explicitly asks for a draft-only knowledge card.

## Handoff Gate

Ask:

`是否将这篇内容转入 Research Knowledge Base Loop？`

Options:

1. `Add to knowledge base`: run the downstream Knowledge Base Loop.
2. `Generate Knowledge Card draft only`: create a draft card, but do not update indexes or beliefs.
3. `Skip for now`: leave the package in the thesis-reader state only.

## Paper Knowledge Package

Use the template in `references/output-templates.md`.

## Safety Rule

The Thesis Reader may recommend knowledge-base links, but it must not silently modify long-term indexes, concept maps, or research beliefs. Those belong to `research-knowledge-base` and require user approval.
